MediaSim

A CLI tool and GUI app to calculate the similarity of images and videos.

Install on macOS / Linux

curl -fsSL https://vegidio.github.io/mediasim/install.sh | sh

Install on Windows

irm https://vegidio.github.io/mediasim/install.ps1 | iex

Options

Both installers accept the same options. By default they install both the CLI and the GUI.

  • CLI only: set MEDIASIM_INSTALL=cli (or pass --cli to install.sh).
  • GUI only: set MEDIASIM_INSTALL=gui (or pass --gui to install.sh).
  • Pin a version: set MEDIASIM_VERSION=<tag> (default: latest).

Example — install only the CLI on macOS / Linux

curl -fsSL https://vegidio.github.io/mediasim/install.sh | MEDIASIM_INSTALL=cli sh

Example — install only the GUI on Windows

$env:MEDIASIM_INSTALL='gui'; irm https://vegidio.github.io/mediasim/install.ps1 | iex
